## Minimum Requirements

- **Android SDK**: API Level 23 (Android 6.0)
- **Target SDK**: API Level 34 (Android 14)
- **Java**: JDK 11 or higher
- **Build Tools**: Android Build Tools 34.0.0

## Repository Sources

- **Google**: Android libraries and Firebase
- **Maven Central**: Standard Java libraries
- **JitPack**: MPAndroidChart library

All dependencies are managed through Gradle and automatically downloaded during build.
